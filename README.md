# Bookord

### Dev

Running dev

```bash
npm run dev
```

Generating docs to `./docs/index.html`

```bash
npm run doc
```

### Dist

Updating targeted browser version & removing unnecessary polyfills

```bash
npx browserslist@latest --update-db
```

Building for windows to `./dist/Bookord.msi`

```bash
npm run dist-windows
```
